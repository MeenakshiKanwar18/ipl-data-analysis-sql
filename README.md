# IPL Data Analysis — SQL Project

Analyzed IPL match and delivery data using MySQL.

## Dataset
- 1,095 matches
- 260,920 deliveries

## Tools Used
- MySQL Workbench
- Python (pandas, sqlalchemy)

## Key Business Questions Answered
- Team performance and win rates
- Toss impact on match results
- Top batters and bowlers
- Venue analysis
- Season-wise title winners

## Files
- `ipl_queries.sql` — All 15 SQL queries with business questions
